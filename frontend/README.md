# Event Booking Management System

A full-stack Event Booking Management System developed as the Day 19 Capstone Project using Spring Boot, React, MongoDB, and JWT Authentication.

---

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Password Encryption (BCrypt)
- Role-Based Access Control (Admin/User)

### User Features
- View Available Events
- Book Event Tickets
- View My Bookings
- Cancel Booking
- Booking Summary Dashboard

### Admin Features
- Admin Dashboard
- Create Event
- Edit Event
- Delete Event
- View All Events
- View Booking Report
- Recent Events
- Latest Bookings
